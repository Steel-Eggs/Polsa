# Сценарии для проекта «СПУТНИК»

Формат записи для каждой ситуации:
- **Ситуация:** краткое описание реальной жизненной истории.
- **Боль/барьер:** почему человеку трудно решить это самому.
- **Как помогает ассистент:** конкретные действия ИИ.
- **Медиума/формат:** куда можно упаковать кейс (шорт, статья, лендинг и т.д.).
- **Примечания:** дополнительные детали.

(список будет пополняться по мере получения сценариев)
